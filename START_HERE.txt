PHOTO-CAT quick start

Windows:
1. Double-click START_WINDOWS.bat.

macOS / Linux:
1. Open Terminal in the photo-cat folder.
2. Run:
   sh START_UNIX.sh

Beginner workflow:
1. Select Catalog CSV.
2. PHOTO-CAT auto-fills Targets CSV, Output/index folder, and Query index folder.
3. Check the column names. They are case-sensitive.
4. Click Save + run.
