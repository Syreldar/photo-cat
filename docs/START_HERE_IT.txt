PHOTO-CAT avvio rapido

Windows:
1. Doppio click su START_WINDOWS.bat.

macOS / Linux:
1. Apri Terminal nella cartella photo-cat.
2. Esegui:
   sh START_UNIX.sh

Flusso base:
1. Seleziona Catalog CSV.
2. PHOTO-CAT compila automaticamente Targets CSV, Output/index folder e Query index folder.
3. Controlla i nomi colonna. Sono case-sensitive.
4. Clicca Save + run.
