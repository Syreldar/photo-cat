# Assets

This folder contains PHOTO-CAT logo assets used by the README files.

- `photo-cat-logo-colored.png`, default fallback logo.
- `photo-cat-logo-light.png`, logo variant for light GitHub theme.
- `photo-cat-logo-dark.png`, logo variant for dark GitHub theme.

Do not rename these files without updating `README.md` and `README_IT.md`.
